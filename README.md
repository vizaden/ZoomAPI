# ZoomAPI
ELEC3 Group Project

Small demo Node.js project serving a static frontend from `public/`.

## Overview

This repository contains a minimal server (`server.js`) and a static frontend in the `public/` folder to demonstrate a Zoom-related API integration or frontend prototype.

## Files

- `server.js` — Node.js server entry point
- `package.json` — project metadata and scripts
- `public/index.html` — main frontend HTML
- `public/script.js` — frontend JavaScript
- `public/style.css` — frontend styles

## Requirements

- Node.js (v14+ recommended)

## Install

```powershell
npm install
```

## Run

Start the server with:

```powershell
node server.js
# or, if a start script exists:
npm start
```

Open your browser at `http://localhost:3000` (or the port configured in `server.js`).
